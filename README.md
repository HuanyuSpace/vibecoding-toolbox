# vibecoding-toolbox

🤖 **本仓库100%由AI编写，无任何手工编码** - 完全通过 Vibe Coding 方式开发

一个用于存放和管理通过 Vibe Coding 生成的各种实用 Web 工具的集合项目。

## 关于本仓库

vibecoding-toolbox 是一个完全由 AI 驱动的开源项目。从项目构思、代码编写到文档完善，每一行代码、每一个功能都是通过与 AI 对话协作完成的。这不仅是一个工具集合，更是 AI 编程能力的展示。

### 什么是 Vibe Coding？

Vibe Coding 是一种革命性的编程方式，通过自然语言与 AI 协作完成软件开发。开发者只需描述需求，AI 就能理解意图并生成高质量的代码。这种方式大大降低了编程门槛，让创意能够快速转化为现实。

### 为什么选择 100% AI 编写？

- **展示 AI 能力**：证明 AI 可以独立完成完整项目的开发
- **快速迭代**：AI 能够迅速响应需求变化，快速生成新版本
- **降低门槛**：即使没有深厚的编程基础，也能通过 AI 创建工具
- **持续进化**：随着 AI 能力的提升，项目也会不断优化和扩展

## 项目介绍

vibecoding-toolbox 是一个工具集合项目，用于存放通过 AI 辅助开发（Vibe Coding）创建的各种实用 Web 应用。每个工具都是独立的 HTML 文件，可以直接在浏览器中运行，无需复杂的构建过程。

## 当前工具

### 1. 🧠 敏感指数测试 (sensitive-quiz.html)
- **功能**: 用于高度敏感群体的心理测试
- **特点**: 
  - 包含 48 道专业题目
  - 科学的评分系统
  - 详细的结果解释
- **评分范围**: 52-140 分

### 2. ✋ 手势控制应用 (gesture-control.html)
- **功能**: 通过摄像头识别手势与页面互动
- **特点**:
  - 支持 5 种手势识别
  - 实时摄像头预览
  - 直观的交互反馈
- **支持的手势**: 张开手掌、点赞、差评、剪刀手、握拳

## 使用方法

1. 克隆或下载本项目
2. 使用浏览器打开 `index.html` 文件
3. 从主页选择你想使用的工具
4. 或者直接在浏览器中打开对应的 HTML 文件

## 项目结构

```
vibecoding-toolbox/
├── index.html              # 主页 - 工具导航入口
├── sensitive-quiz.html    # 敏感指数测试
├── gesture-control.html   # 手势控制应用
└── README.md              # 项目说明文档
```

## 技术栈

- **HTML5**: 页面结构
- **CSS3**: 样式和动画
- **JavaScript**: 交互逻辑
- **MediaPipe**: 手势识别（手势控制应用）
- **localStorage**: 本地数据存储

## 开发说明

本项目采用 Vibe Coding 方式开发，即通过与 AI 对话协作生成代码。每个工具都是独立的，可以单独使用或扩展。

### 添加新工具

1. 创建新的 HTML 文件
2. 在 `index.html` 中添加工具卡片
3. 更新统计信息（如需要）
